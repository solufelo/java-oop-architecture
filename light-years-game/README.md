# Light Years Game

A space shooter game built with SFML and Box2D.

## Game Features:
- [ ] Game loop and rendering
- [ ] Actor system
- [ ] Player spaceship with movement
- [ ] Shooting mechanics
- [ ] Physics integration (Box2D)
- [ ] Collision detection
- [ ] Health system
- [ ] Particle effects
- [ ] Asset management

## Technologies:
- **SFML** - Graphics library
- **Box2D** - Physics engine
- **CMake** - Build system

## Build Instructions:
```bash
mkdir build
cd build
cmake ..
cmake --build .
```
